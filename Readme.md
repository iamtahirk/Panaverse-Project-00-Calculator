
# Panaverse Project - Calculator

Developed a simple command line calculator using TypeScipt, Node.js and Inquirer.

## Deployment

Clone the repo and run following command.

```bash
  npm i
```


## How to Run

Run the following command to check this out.

```bash
  npx tscalc-cli
```
